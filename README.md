# 積立投資シミュレーター

Vue.jsを用いた積立投資シミュレーションWebアプリ。毎月の積立・期間・利回りを入力するだけで、資産推移をグラフと表で視覚化します。

## 機能

- **投資条件の設定**: 初期投資額・毎月積立額・投資期間・期待利回り・複利/単利の選択
- **シミュレーション結果**: 総投資額・最終資産額・運用益をリアルタイム表示
- **資産推移グラフ**: 年次折れ線グラフで資産と投資額の推移を可視化
- **年次データ表**: 各年の累計投資額・運用益・評価額・利益率を一覧表示
- **入力バリデーション**: 無効な値に対して日本語エラーメッセージを表示

## 技術構成

- **フレームワーク**: Vue.js 3 + TypeScript (Vite)
- **UIライブラリ**: Vuetify 3 (ダークテーマ, ブルー/パープル系)
- **状態管理**: Pinia
- **グラフ**: Chart.js + vue-chartjs
